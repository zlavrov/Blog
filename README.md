# Blog
Blog 

- Symfony 6
- Vue 3
- Node 20

# How to get

- git clone ...

# How to start

- composer install
- Set your preferences in the .env file
- php bin/console doctrine:database:create
- php bin/console doctrine:migrations:migrate
- npm install --force or npm install && npm run dev
- npm run dev or npm run watch
- symfony serve -d
- Enjoy
